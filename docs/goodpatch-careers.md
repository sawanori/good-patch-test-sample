# Goodpatch Careers Site

Source: https://careers.goodpatch.com/

## 会社概要

**Goodpatch**はデザインに特化した企業で、約150名以上の社員が在籍。このサイトはGoodpatch初の公式採用サイトであり、企業的な体裁よりも「本音」を重視した構成になっている。

**キーメッセージ:** 「これは、Goodpatchが初めてつくった採用サイトです。デザイン会社として、多くのデザイナーに出会いたい。同時に、デザインの力を信じるビジネスプロフェッショナルやエンジニアにも出会いたい。」

## UI/UXデザイン分析

### レイアウト構造
- 明確な階層構造を採用
- ナビゲーションは上部に固定
- 大型ヒーローイメージ（「あなたのやりたいことは、デザインかもしれない」）から開始
- セクション形式で情報を段階的に提示

### カラースキーム
- **背景:** 白ベース（クリーン、プロフェッショナル）
- **テキスト:** 深い色での高いコントラスト
- **アクセント:** ロゴはホワイトとダークグレイのツートーン表示（背景に応じて視認性を確保）

### タイポグラフィ
カスタムフォントシステムを採用：
```css
--font-headings-default: システムフォント（San Francisco, Segoe UI等）
--font-base-default: 同上
```
プラットフォーム間での一貫性を保証

### アニメーション機能
- サイト上部に「アニメーション ON/OFF」トグルを設置
- ユーザーが視覚効果を制御可能
- アクセシビリティとパフォーマンスを両立

### ナビゲーション構造
6つのメインセクション：
1. About
2. Culture
3. Job
4. People
5. Workstyle
6. Process

### インタラクション要素
**主要CTA:**
- 「Entry 38職種」（大型ボタン）
- 「新卒27卒」へのリンク
- YouTube、note、Spotify等への関連コンテンツリンク

### コンポーネント設計
- **カード型レイアウト:** 職種紹介（UI/UXデザイナー、エンジニア等）を視覚的に分類
- **セクションアイコン:** 各カテゴリにSVGアイコンを配置して視認性向上
- **Quicklink統合:** プリフェッチ機能で次ページへの遷移を最適化

### レスポンシブデザイン
- スマートフォン対応（`hidden sm:block`クラスで表示制御）
- モバイルファーストのレスポンシブ実装

### メディア統合
- YouTube動画
- note記事
- Spotify音声コンテンツ
- 階層的に展開される構成
